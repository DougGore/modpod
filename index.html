MODpod

When you hear music on the Internet it is usually streamed from a compressed audio file. What if your web browser could perform live music instead of streaming it? Using the forthcoming W3C WebAudio API audio can be dynamically synthesised right here in your web browser. This project will replay "tracker style" music via your browser.

Supported browsers:
Chrome - Safari 6 - Safari (iOS) 6
Awaiting support:
Internet Explorer - Firefox (forthcoming) - Chrome (Android)
